# Amazon Clone

A polished front-end replica of the Amazon shopping experience, built with HTML, CSS, and JavaScript. This project demonstrates responsive layout, interactive UI components, and product listing design inspired by the Amazon storefront.

## 🚀 Project Overview

This Amazon Clone is designed as a landing page style web app that showcases:
- Product cards with images, ratings, and prices
- A clean and responsive layout for desktop and mobile
- Modern styling inspired by Amazon's shopping interface
- A simple hero section and navigation bar

## 🎯 Features

- Responsive design using CSS layout techniques
- Product listing grid with hover states
- Styled header and search bar area
- Footer section for site links and info
