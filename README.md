# Amazon Clone

A polished front-end replica of the Amazon shopping experience, built with HTML and CSS only. This project is a static page meant for learning layout and styling, with a design inspired by the Amazon storefront.

![alt text](<Screenshot 2026-06-16 235806-1.png>)

## 🚀 Project Overview

This Amazon Clone is designed as a landing page style web app that showcases:
- Product cards with images, ratings, and prices
- A clean and responsive layout for desktop and mobile
- Modern styling inspired by Amazon's shopping interface
- A simple hero section and navigation bar

## 🎯 Features

- Responsive design using CSS layout techniques
- Product listing grid with hover states
- Styled header and search bar area
- Footer section for site links and info



## 🛠️ Built With

- HTML
- CSS

## 📁 Project Structure

- `index.html` — main page layout
- `style.css` — styling and responsive design
- `images/` — visual assets used for products and layout

## ✅ How to Run

1. Open `index.html` in your browser.
2. Or serve the folder using a local development server.

## 💡 Notes

- Customize the product cards, images, and colors to make the store your own.
- Keep this project as a clean HTML/CSS learning exercise.
- In the future, you can add JavaScript to enable a cart, search, or interactive product details.
